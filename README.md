# デモの始め方

**前提条件**: Unityをインストール済み

Unity Editorバージョン: 6000.0.74f1

## 1. GithubDesktopのインストール（初回のみ）

**[ダウンロードリンク](https://desktop.github.com/download/)**

## 2. リポジトリをクローンする

下記のサイトなどを参考にリポジトリをクローンする
https://qiita.com/S1riuXs/items/87c9d1bade8ae697c08d

クローンするリポジトリ

```
https://github.com/Gifu-University-Digital-Criation-Club/Unity-2DRPG-Demo.git
```

## 3. Unityで開く

Unity Hubのプロジェクトタブから右上の追加>ディスクから加えるでクローンしたものがあるフォルダを選択する。

## 4. 改良する

あとは自由に変更できるので、ステージの形を変えたり、敵の数を増やす、コードを書き替えるなどしてみてください！
